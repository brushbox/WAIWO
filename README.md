# What Am I Working On (WAIWO)

This is intended to be a simple macOS app.

- it finds the top incomplete TODO item from my current/most recent Obsidian daily note
- it displays this in a floating always-on-top window
- the window avoids the mouse pointer (it stays out of the way of the work being done)
- it allows me to disable it (maybe it hooks into the "Focus" settings?)
- it can optionally have a fading list of the upcoming todos.
